# O que é a _toolchain_ #

_Toolchain_ ou "cadeia de ferramentas" em tradução literal consiste no conjunto básico de ferramentas sobre a qual a nova distribuição vão ser construída.
Ela (a toolchain) é criada em um sistema hospedeiro e geralmente possibilita a compilação cruzada, o que significa que além do sistema a ser construído


# Uso #

Não há nada a princípio para instalar. Como explicado, a _toolchain_ vai servir como base para a construção do novo sistema; para tal, o processo consiste nos seguintes passos:
  * Descomprimir o arquivo .bz2 em um diretório vazio de sua escolha;
  * Criação dos dispositivos necessários para suporte do sistema;
